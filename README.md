# AI-Assisted Note Summary

<img width="1920" height="1080" alt="screenshot8" src="https://github.com/user-attachments/assets/98322772-74b4-45a9-884c-0548c7b5f22a" />
A Python-based tool for automating the summarization of study notes using AI assistance. This script leverages AI models to process and condense lengthy notes into concise summaries, making it easier to review and retain key information.


## Features

  - AI-Powered Summarization: Integrates with AI APIs to generate intelligent summaries.
  - The tool can be use offline

## Prerequisites

  - Python 3.8 or higher
  - An API AI Model 
  - Required Python libraries (see Installation)

## Installation

  1. Clone the repository:
  ```
    git clone https://github.com/apitongcm/AI-assisted-note-summary.git
  ```

  2. Navigate to the project directory:
  ```
  cd AI-assisted-note-summary
  ```

  3. Install the required dependencies:
  ```  
  pip install -r requirements.txt
  ```
## Usage
  ```
  # Extract data from pdf to docx
  python3 study-summary-automation.py extract input.pdf

  # Summarize data from docx input
  python3 study-summary-automation.py summarize input.docx
 ```
